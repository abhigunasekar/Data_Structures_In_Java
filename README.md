# Algorithms
- Sorting Algorithms
  1. Bubble Sort - O(n^2)
  2. Insertion Sort - O(n^2)
  3. Selection Sort - O(n^2)
  4. Shell Sort - O(n^2)
  4. Merge Sort - O(nlogn)
  5. Counting Sort - O(n)
  6. Radix Sort - O(n)
- Searching 
  1. Linear Search
  2. Binary Search
